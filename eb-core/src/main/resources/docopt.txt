Usage:
  eb <cmdSpec> ... [ -h | -v | -vv ]
  eb activitytypes
  eb help <activitytype>
  eb ( script <script> ) ...
  eb ( activity <activity> ) ...

Options:
  -h                               Show help.
  -v                               Show version, short form.
  -vv                              Show version, long form, with artifact coordinates.
  -l <logfile>, --log <logfile>    Set the script logfile  [default: bm.log]
